    <ul class="md-number-list">
      <li>
        だれかの作品を 勝手に使わない（著作権）
        <ul class="md-point-list1"> 
          <li> アニメ・マンガ・ゲームの絵・音・音楽を許可なしで使わない </li>
          <li> ネットの写真・イラスト・プログラムをそのまま使わない </li>
          <li> 自分で作ったもの・「使って良いよ」と書いてある素材はOK! </li>
        </ul>
      </li>
      <li>
        人を傷つける・怖がらせることはしない
        <ul class="md-point-list1">
          <li> なぐる、ころす、いじめるなどの表現をしない </li>
          <li> いやな言葉で人をばかにしない </li>
        </ul>
      </li>
      <li>
        名前や住所を出さない（プライバシー）
        <ul class="md-point-list1">
          <li> 学校名、住所、電話番号は書かない </li>
          <li> ニックネームなどはOK! </li>
        </ul>
      </li>
      <li>
        誰かをきらったり、なかまはずれにしない
        <ul class="md-point-list1">
          <li> 国や人との違いをばかにしない </li>
          <li> 「〇〇はだめ！」と決めつけない </li>
        </ul>
      </li>
      <li>
        あぶないことを、すすめない
        <ul class="md-point-list1">
          <li> マネするとケガしそうなことは、そのまま出さない </li>
          <li> ひつようなら「大人といっしょにね」と書こう </li>
        </ul>
      </li>
    </ul>
